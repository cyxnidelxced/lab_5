1. Responsive Background Image

This example demonstrates a responsive background image using CSS properties. The image's position and size change based on the viewport width:

  • On large screens (>768px): The image is centered and uses "cover" to fill the container
  • On medium screens (768px and below): The image is positioned at the top center
  • On small screens (480px and below): The image uses "contain" for the size and is centered at the top

The CSS properties used are:

  • background-image
  • background-position
  • background-size

Try resizing your browser to see how the image adapts to different viewport sizes.

2. Object-fit and Object-position Properties

This example demonstrates how "object-fit: cover" combined with "object-position" creates responsive images that scale without distortion. The image maintains its aspect ratio while filling the container completely, and the visible portion changes based on screen size:

  • On large screens: Image is centered (object-position: center)
  • On medium screens (≤768px): Image shows the top portion (object-position: top)
  • On small screens (≤480px): Image shows the left portion (object-position: left)

This approach prevents stretching or distortion while ensuring the most important parts of the image remain visible at different screen sizes. Try resizing your browser to see the effect.

3. Resolution Switching with srcset and sizes

This example demonstrates resolution switching using the srcset and sizes attributes. The browser automatically selects the appropriate image based on the viewport width and device pixel ratio.

  • Small image (500px wide) - For smaller viewports
  • Medium image (1000px wide) - For medium viewports
  • Large image (2000px wide) - For large viewports or high-density screens

The sizes attribute tells the browser how much of the viewport width the image will occupy at different breakpoints.

4. Art Direction with picture and source

This example demonstrates art direction using the picture and source elements. Different images with different orientations are displayed based on the viewport width:

  • On narrow screens (<600px): A portrait-oriented image is shown (better suited for mobile)
  • On wider screens (≥600px): A landscape-oriented image is shown (better suited for desktop)

This technique is used when you need to change the composition, crop, or orientation of an image rather than just its resolution to better suit different device layouts.

5. Scalable Vector Graphics (SVG)

This example shows a scalable SVG image. Unlike bitmap images (like JPG or PNG), SVG images are vector-based and can scale to any size without losing quality. They're perfect for logos, icons, and simple illustrations. The SVG below will maintain its sharpness regardless of the display size or screen resolution. Try zooming in or viewing this page on different devices to see how the SVG remains crisp.
